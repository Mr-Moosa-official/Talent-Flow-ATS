# **App Name**: TalentFlow ATS

## Core Features:

- Job Requisition Management: Create, edit, and manage job requisitions with detailed descriptions, required skills, and salary ranges.
- Candidate Profile Creation: Create and update candidate profiles with resume uploads, contact information, work experience, and skills. Automatically parse resume data to pre-fill profile fields using AI.
- Applicant Tracking: Track applicants through custom pipeline stages (e.g., Application, Phone Screen, Interview, Offer) with automated notifications and status updates. A generative AI tool recommends next steps for each candidate based on their profile and stage in the pipeline.
- Search and Filter: Enable powerful search and filter capabilities to quickly find candidates based on keywords, skills, experience, location, and other criteria.
- Review System: The review section can have reviews that can be about different categories like project and/or the course and all the others. Review rating can vary from 1 to 5 and reviews are anonymous.
- Project Overview: Showcasing projects, the main parts are overview and short project description and rating which users left.
- Course Listing: A scrollable carousel element in which courses appear and can be rated.

## Style Guidelines:

- Primary color: Cornflower blue (#6495ED) to evoke a sense of professionalism and trust.
- Background color: Off-white (#F5F5F5) for a clean, modern look.
- Accent color: Sea green (#3CB371) for highlighting key actions and elements.
- Body and headline font: 'Inter', sans-serif, providing a modern and readable interface.
- Use simple, professional icons to represent different actions and data points.
- Maintain a clean, well-organized layout with clear hierarchy and intuitive navigation.
- Use subtle animations and transitions to enhance the user experience and provide feedback on actions.
